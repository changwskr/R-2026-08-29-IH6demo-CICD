
마이크로서비스 스프링클라우드 핵심

● 개발패턴
    - 핵심마이크로서비스 패턴 : 스프링부트
    - 구성관리 : 스프링클라우드 컨피그
    - 비동기 메시지 : 스프링클라우드 스트림

● 라우팅 패턴
    - 서비스디스커버리 : 스프링클라우드 / 넷플릭스 유레카

● 클라이언트 회복성 패턴
    - 클라이언트 측 부하분산 : 스프링클라우드 / 넷플릭스 리본
    - 회로 차단기 패턴 : 스프링클라우드 / 넷플릭스 히스트릭스
    - 폴백 패턴 : 스프링클라우드 / 넷플릭스 히스트릭스
      폴백은 장애가 난 사이트에 대한 대응 경로가 있는가
    - 벌크헤드 패턴 : 스프링클라우드 / 넷플릭스 히스트릭스
      오작동하는 서비스에 대한 대응 정책

● 빌드배포 패턴
    - 지속적 통합
    - 코드형 인프라스트럭처 : 도커
    - 불변서버 : 도커
    - 피닉스 서버 : 도커

● 로깅 패턴
    - 로그 상관관계 : 스프링클라우드 솔루스

    - 로그수집 : 스프링클라우드 솔루스
    - 마이크로서비스 추적 : 스프링 클라우드 솔루스 / 집킨

● 보안패턴
    - 인가 : 스프링 클라우드 시큐리티
    - 인증 : 스프링 클라우드 시큐리티
    - 자격증명 관리와 전파 : 스프링 클라우으 시큐리티
